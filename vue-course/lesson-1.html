Введение в Vue.js

Что такое Vue.js?

Vue.js — это прогрессивный JavaScript фреймворк для создания пользовательских интерфейсов. Он отличается лёгкостью изучения и гибкостью использования.

Особенности Vue.js

  • Реактивность — данные автоматически обновляют интерфейс
  • Компонентный подход — переиспользуемые блоки кода
  • Лёгкий порог входа — быстрое освоение
  • Гибкость — можно использовать как библиотеку или полноценный фреймворк

Установка через CDN

Самый быстрый способ начать — подключить Vue через CDN:

<script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

Первое приложение

Создадим простое приложение "Hello Vue":

<div id="app">
  {{ message }}
</div>

<script>
  const app = Vue.createApp({
    data() {
      return {
        message: 'Hello Vue!'
      }
    }
  })
  app.mount('#app')
</script>

Ключевые концепции

  1. Экземпляр Vue — корневой объект приложения
  2. Реактивность — автоматическое обновление UI при изменении данных
  3. Шаблоны — HTML с Vue-синтаксисом
  4. Директивы — специальные атрибуты для реактивного поведения

Практическое задание

Создай HTML-файл с:

  1. Подключённым Vue 3 через CDN
  2. Заголовком "Моё первое Vue приложение"
  3. Кнопкой, которая меняет текст при клике
