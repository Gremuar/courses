Vue Instance

Создание экземпляра Vue

Приложение Vue начинается с создания экземпляра. Это корневой объект, который управляет всем приложением.

const app = Vue.createApp({
  // опции приложения
})
app.mount('#app')

Объект Data

В data() хранятся реактивные данные приложения. Vue автоматически отслеживает их изменения.

data() {
  return {
    username: 'Ivan',
    age: 25,
    isActive: true,
    items: ['a', 'b', 'c']
  }
}

Methods (Методы)

Мethods содержат функции, которые можно вызывать из шаблона. Доступ к данным через this.

methods: {
  greet() {
    return `Привет, ${this.username}!`
  },
  changeName(newName) {
    this.username = newName
  }
}

Computed (Вычисляемые свойства)

Computed — это свойства, которые автоматически вычисляются на основе зависимостей. Результат кэшируется.

computed: {
  fullName() {
    return this.firstName + ' ' + this.lastName
  },
  isAdult() {
    return this.age >= 18
  }
}

Как работает реактивность?

Vue отслеживает изменения в data и автоматически обновляет DOM. Это называется реактивностью.

// При изменении count, DOM обновится автоматически
data() {
  return { count: 0 }
}

<button @click="count++">{{ count }}</button>
