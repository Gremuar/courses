Директивы

Что такое директивы?

Директивы — это специальные атрибуты с префиксом v-, которые добавляют реактивное поведение к HTML-элементам.

v-if / v-else — Условный рендеринг

Отображают или скрывают элемент в зависимости от условия.

<div v-if="isLoggedIn">Добро пожаловать!</div>
<div v-else>Войдите в систему</div>

v-show — Скрытие элемента

Скрывает элемент через CSS display, но не удаляет из DOM.

<div v-show="isVisible">Показывается/скрывается</div>

v-for — Цикл по массиву

Позволяет рендерить список элементов.

<ul>
  <li v-for="(item, index) in items" :key="index">
    {{ item.name }}
  </li>
</ul>

v-bind — Динамические атрибуты

Связывает атрибут элемента с выражением. Сокращение — двоеточие (:).

<img :src="imageUrl" :alt="imageAlt">
<button :disabled="isDisabled">Нажми</button>

v-on — Обработка событий

Подписывается на события элемента. Сокращение — @.

<button @click="handleClick">Нажми</button>
<form @submit.prevent="submitForm">...</form>

v-model — Двустороннее связывание

Создаёт двустороннюю связь между input и данных.

<input v-model="username">
<textarea v-model="message"></textarea>
<select v-model="selected">...</select>

Модификаторы

Модификаторы изменяют поведение директив.

// .trim — обрезает пробелы
<input v-model.trim="text">

// .number — преобразует в число
<input v-model.number="age">

// .stop — останавливает всплытие
<button @click.stop="handleClick">
