Компоненты

Что такое компонент?

Компонент — это переиспользуемый блок Vue с собственным шаблоном, данными и логикой.

Создание компонента

const MyComponent = {
  template: '<div>Мой компонент</div>'
}

Регистрация компонента

// Глобальная регистрация
app.component('my-component', MyComponent)

Props — передача данных

Props позволяют передавать данные от родителя к ребёнку.

// Дочерний компонент
props: {
  title: String,
  count: {
    type: Number,
    default: 0
  }
}

// Использование
<my-component title="Привет" :count="5"></my-component>

Events — передача событий

Дочерний компонент может сообщать родителю о действиях через $emit.

// В дочернем компоненте
this.$emit('update', newValue)

// В родительском компоненте
<my-component @update="handleUpdate"></my-component>

Slots — слоты

Слоты позволяют передавать контент внутрь компонента.

// Родитель
<my-card>
  <h2>Заголовок</h2>
</my-card>

// Дочерний (MyCard)
<div class="card">
  <slot></slot>
</div>
