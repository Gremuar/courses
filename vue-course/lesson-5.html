Computed & Watch

Computed — вычисляемые свойства

Computed — это свойства, которые автоматически вычисляются на основе зависимостей. Результат кэшируется.

computed: {
  fullName() {
    return this.firstName + ' ' + this.lastName
  },
  isAdult() {
    return this.age >= 18
  },
  totalPrice() {
    return this.items.reduce((sum, item) => sum + item.price, 0)
  }
}

Преимущества computed

  • Кэширование — результат не пересчитывается без необходимости
  • Автообновление — зависимости отслеживаются автоматически
  • Чистота кода — логика вынесена отдельно

Watch — наблюдатели

Watch позволяет выполнять код при изменении определённого свойства.

watch: {
  searchQuery(newVal, oldVal) {
    console.log(`Изменено с ${oldVal} на ${newVal}`)
    this.fetchData()
  },
  // Глубокий наблюдатель
  options: {
    handler() { /* ... */ },
    deep: true
  }
}

Когда что использовать?

Computed Watch
Вычисление значений Асинхронные операции
Зависимости от данных Выполнение при изменении
Возвращает значение Выполняет side effects
