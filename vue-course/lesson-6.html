Жизненный цикл компонентов

Этапы жизненного цикла

Каждый компонент Vue проходит через несколько этапов: создание, монтирование, обновление и удаление.

Создание → Монтирование → Обновление → Удаление

Основные хуки

created

Вызывается после создания экземпляра. Данные и методы уже доступны.

mounted

Вызывается после монтирования в DOM. DOM готов к работе.

updated

Вызывается после обновления DOM из-за реактивных изменений.

unmounted

Вызывается после удаления компонента из DOM.

export default {
  created() {
    console.log('Компонент создан')
  },
  
  mounted() {
    console.log('DOM готов')
    // Получение данных, подписки
  },
  
  updated() {
    console.log('Компонент обновлён')
  },
  
  unmounted() {
    console.log('Компонент удалён')
    // Очистка таймеров, подписок
  }
}

Практический пример

export default {
  data() {
    return { users: [] }
  },
  
  created() {
    this.fetchUsers()
  },
  
  methods: {
    async fetchUsers() {
      this.users = await fetch('/api/users').then(r => r.json())
    }
  }
}

Практическое задание

Добавь таймер, который запускается при монтировании и очищается при удалении.
