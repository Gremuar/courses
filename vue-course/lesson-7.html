Vue CLI и Vue Router

Создание проекта с Vite

Vite — современный инструмент сборки, рекомендуемый для новых проектов Vue.

# Создание проекта
npm create vue@latest
# или
npm create vite@latest my-vue-app -- --template vue

Структура проекта

my-app/
├── src/
│   ├── assets/
│   ├── components/
│   ├── App.vue
│   └── main.js
├── index.html
└── package.json

Vue Router

Vue Router позволяет создавать одностраничные приложения с несколькими видами.

Установка

npm install vue-router

Настройка роутера

import { createRouter, createWebHistory } from 'vue-router'
import Home from '@/views/Home.vue'
import About from '@/views/About.vue'

const routes = [
  { path: '/', component: Home },
  { path: '/about', component: About },
  { path: '/user/:id', component: User }
]

const router = createRouter({
  history: createWebHistory(),
  routes
})

export default router

Использование в App.vue

<router-link to="/">Главная</router-link>
<router-link to="/about">О нас</router-link>
<router-view></router-view>

Динамические маршруты

// Получение параметра
this.$route.params.id
// или в Composition API
const route = useRoute()
route.params.id
