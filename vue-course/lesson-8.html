Pinia — управление состоянием

Что такое Pinia?

Pinia — официальная библиотека для управления состоянием в Vue. Она заменила Vuex и стала рекомендуемым решением.

Установка

npm install pinia

Настройка в main.js

import { createPinia } from 'pinia'
const pinia = createPinia()
app.use(pinia)

Создание Store

import { defineStore } from 'pinia'

export const useUserStore = defineStore('user', {
  state: () => ({
    name: '',
    email: '',
    isLoggedIn: false
  }),
  
  getters: {
    greeting: (state) => `Привет, ${state.name}!`
  },
  
  actions: {
    login(userData) {
      this.name = userData.name
      this.email = userData.email
      this.isLoggedIn = true
    },
    logout() {
      this.name = ''
      this.email = ''
      this.isLoggedIn = false
    }
  }
})

Использование в компоненте

import { useUserStore } from '@/stores/user'

export default {
  setup() {
    const userStore = useUserStore()
    
    return { 
      userStore,
      name: userStore.name
    }
  }
}

Основные понятия

  • state — данные (как data в компоненте)
  • getters — вычисляемые свойства (как computed)
  • actions — методы для изменения состояния

Практическое задание

Создай store для списка задач с добавлением, удалением и фильтрацией.
