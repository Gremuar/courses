Composition API

Введение

Composition API — новый подход в Vue 3 для организации логики компонентов. Он предлагает более гибкую структуру кода.

script setup

Современный синтаксис, делающий код короче и понятнее.

<script setup>
import { ref, computed } from 'vue'

const count = ref(0)
const doubled = computed(() => count.value * 2)

function increment() {
  count.value++
}
</script>

<template>
  <button @click="increment">
    {{ count }} x 2 = {{ doubled }}
  </button>
</template>

ref vs reactive

ref — для примитивов

const count = ref(0)
count.value++  // .value для доступа

reactive — для объектов

const state = reactive({
  user: { name: 'Ivan' },
  items: []
})
state.user.name = 'Petya'  // без .value

Composables

Переиспользуемая логика в виде функций.

// useCounter.js
import { ref } from 'vue'

export function useCounter() {
  const count = ref(0)
  const increment = () => count.value++
  const decrement = () => count.value--
  
  return { count, increment, decrement }
}

Использование composable

importuseCounter } from '@/composables/useCounter'

const { count, increment, decrement } = useCounter()

Lifecycle в Composition API

import { onMounted, onUnmounted } from 'vue'

onMounted(() => {
  console.log('Монтирован')
})

onUnmounted(() => {
  console.log('Размонтирован')
})
