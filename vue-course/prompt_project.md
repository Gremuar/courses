# Prompts for Vue.js Course Project

## Сводка проекта
Обучающий курс по Vue.js с 10 уроками, встроенными тестами и прогресс-трекером.
Написан ИИ-моделью MiniMax M2.5 Free.

---

## Промпт 1: Начальное создание курса

**Пользователь:**
> Представь что ты крутой учитель по VueJS. Тебе нужно подготовить обучающий курс. Твоя задача обучить меня этому фреймворку. Я знаю азы js и работы браузера. Составь план обучения разбей его на этапы. План запиши в отдельный файл, документируй прохождение каждого этапа с подтверждением пользователя. Сгенерируй на каждый этап небольшой тест для закрепления. Сделай сводную таблицу html для отображения текущего состояния прохождения обучения.
> На основе этого промпта сгенерируй агентские правила, скилы.
> Для обучения доступна установка любых инструментов в рамках достижения текущей задачи.

**Результат:**
- Создан `PLAN.md` с 10 этапами
- Создан `progress.html` - таблица прогресса
- Создан JSON структура курса
- Созданы материалы этапов в Markdown
- Созданы HTML тесты для каждого этапа
- Созданы агентские правила и скилы

## Промпт 3: Переработка курса (основной)

**Пользователь:**
> Все уроки должны быть в html, а не в md. И главное меню в progress.html нужно сделать более красочно по дизайну и адаптивно под мобильные устройства. Так же должна быть последовательность уроков, общий план в котором в конце урока идет тест на закрепление. Все должно быть последовательно с минимумом кнопок. И из любого урока должен быть доступ к оглавлению.

**Ответы на уточняющие вопросы:**
- Стиль меню: **Современный Colorful** (яркие цвета, градиенты)
- Тесты: **Встроенные в урок** (в конце каждого урока)
- Навигация: **Боковое меню** (раскрывается по кнопке ☰)

**Результат:**
- Переработан `progress.html` в `index.html` с красочным дизайном
- Все уроки переведены в HTML формат (lesson-1.html - lesson-10.html)
- Тесты встроены в каждый урок
- Добавлено боковое меню с оглавлением
- Создан единый CSS файл со стилями
- Адаптивный дизайн для мобильных устройств

## Особенности реализации

### Дизайн
- Градиентный header (фиолетовый → розовый → оранжевый)
- Тёмная тема с яркими акцентами
- Анимации при загрузке и взаимодействии

### Функциональность
- Прогресс сохраняется в localStorage
- Боковое меню (☰) с оглавлением всех уроков
- Встроенные тесты (5 вопросов, проверка после прохождения)
- Последовательная навигация между уроками
- Адаптивность для мобильных устройств

### Тесты
- 5 вопросов на каждый урок
- Соответствие материалу урока
- Показ правильных ответов после проверки
- Минимум 3/5 для прохождения

---

## Как использовать курс

1. Открыть `index.html` в браузере
2. Нажать "Начать" на первом уроке
3. Изучить материал урока
4. Пройти встроенный тест
5. После прохождения - перейти к следующему уроку
6. Отслеживать прогресс в главном меню
